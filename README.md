# PetriNet

Редактор и симулятор сетей Петри на Rust (`eframe/egui`).

## Структура проекта

- `src/main.rs` — точка входа приложения.
- `src/ui/` — интерфейс, инструменты редактирования графа, окна свойств, отладка.
- `src/model.rs` — модель сети Петри (места, переходы, дуги, таблицы).
- `src/sim/` — движок имитации.
- `src/io/` — загрузка/сохранение (`legacy .gpn`, `gpn2`).
- `src/formats/` — дополнительные форматы.
- `tests/` — интеграционные тесты импорта/экспорта и симуляции.
- `assets/` — ресурсы приложения (иконка и пр.).

## Сборка

```bash
cargo run
cargo test
cargo build --release
```

## EXE-сборка (Windows)

- `build_exe.ps1` — обычная release-сборка.
- `build_portable_exe.ps1` — portable сборка (static CRT), выходной файл: `PetriNet-0.3.0.exe`.

## Форматы файлов

- `legacy .gpn` — импорт/экспорт для совместимости с NetStar.
- `gpn2` — внутренний формат проекта.

Подробно по формату: `docs/gpn_format.md`.
